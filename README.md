# yadisk-gui
## Setting up 
### Cloning repository
```
git clone https://github.com/kabachoke/yadisk-gui.git
cd yadisk-gui
```
### Setting up virtual environment
```
python -m venv venv
source ./venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```
### Run
`python main.py`
